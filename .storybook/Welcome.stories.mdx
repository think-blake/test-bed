# Welcome

The Think Standard Library is the start of a design system. It includes styles, components, and patterns for various UI elements within your site. This library is intended to be used as a reference and guide for the design and development teams when enhancing the web experience.

<img alt="Random tech image" src="https://placeimg.com/1000/300/tech" />

<br />

## What's Included

### Primitives

The foundational styles and components that are used heavily throughout the UI library. This includes colors, typography, spacing, and more.

### Inputs & Controls

These components allow users to input data and perform actions within the product. This includes things like buttons, text inputs, selects, and other controls.
